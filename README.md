# 📚 Smart Library System (OOP)

## Overview
This project is an Object-Oriented Programming (OOP) solution for a Smart Library System developed as a university assignment. The system automates library operations such as book borrowing, returning, inventory management, and borrowing history tracking.

## Features
- User management using inheritance
- Different user roles:
  - Student
  - Faculty
  - Librarian
- Book borrowing and returning
- Inventory tracking
- Borrowing history management
- Role-based permissions (only librarians can add or remove books)
- Polymorphism for user-specific functionalities

## OOP Concepts Used
- Classes & Objects
- Inheritance
- Polymorphism

## Class Structure
- User (Base Class)
- Student
- Faculty
- Librarian
- Book
- systemflow

## Learning Outcomes
This project demonstrates how Object-Oriented Programming principles can be applied to design a real-world library management system with reusable and maintainable code.
